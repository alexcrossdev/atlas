# Atlas

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

**Atlas** is a fast, lightweight terminal emulator and tool suite designed for performance and extensibility.

---

## Features
- Uses the **Bash** shell by default
- Integrated **Orion** text editor

---

## Requirements

- Rust toolchain (**Cargo**)

---

## Build & Run
```bash
cargo run
```
